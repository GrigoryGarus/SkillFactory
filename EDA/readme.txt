Основные цели и задачи проекта:  отследить влияние условий жизни учащихся на их успеваемость score,
провести разведывательный анализ данных и составить отчёт по его результатам.

Информация о данных: 29 столбцов с данными о учениках, есть выбросы и пустые значения.

Этапы работы над проектом: написание функций для удаления выбросов, заполнение пропусков на среднее, среднее значение score для каждой колонки для анализа, удаление пропусков, убедиться что нет пропусков, построение pairplot с score.
